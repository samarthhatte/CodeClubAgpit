**CodeClubAgpit** is the official community application for the Computer Science department's student organization at AG Patil Institute of Technology (AGPIT). It serves as a centralized hub for event notifications, resources, and student collaboration.

## ✨ Features

- **User Authentication:** Secure login/signup using Firebase Auth (Email/Password & Google Sign-in).
- **Event Management:** Real-time updates on upcoming workshops, hackathons, and seminars.
- **Resource Sharing:** Access to study materials and notes managed via Cloudinary storage.
- **Student Directory:** Connect with fellow AGPITians and lead members.
- **Admin Dashboard:** Specific controls for club leads to post updates and manage content.

## 🛠 Tech Stack

- **Language:** Java
- **UI:** XML (Material Design)
- **Backend:** Firebase (Firestore, Auth, Cloud Messaging)
- **Media Storage:** Cloudinary
- **Architecture:** MVVM / MVC
